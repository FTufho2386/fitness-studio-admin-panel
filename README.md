# Fitness Studio Admin Panel

Админ-панель для управления фитнес-студией. Проект разработан на Python.

## Функционал
* Управление расписанием тренировок
* Запись клиентов на занятия и отмена записей
* Управление базой данных клиентов и абонементов

## Стек технологий
* Python 3
* FastAPI 
* PostgreSQL
